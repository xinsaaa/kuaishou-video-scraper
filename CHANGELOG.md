## [1.0.0] - 2025-12-17

### 新增
- 🎉 首次发布
- ✨ 基于 PyQt6 的图形用户界面
- 🚀 异步并发爬取，支持自定义并发数
- 📊 支持 Excel 文件批量导入和导出
- 🔄 智能重试机制，提高爬取成功率
- 📝 实时进度显示和详细日志
- 🛡️ 完善的错误处理和异常记录
- 📦 一键打包为可执行文件

### 支持的功能
- 快手视频信息爬取
  - 视频标题、作者信息
  - 点赞、评论、分享、播放数据
  - 发布时间等元数据
- 多种链接格式支持
  - 短链接：`v.kuaishou.com`
  - 完整链接：`www.kuaishou.com/short-video/`
  - 移动端链接：`m.kuaishou.com`
- 批量处理
  - Excel 文件导入
  - 自动去重和格式化
  - 结果导出为 Excel 格式

### 技术特性
- 异步 HTTP 请求（aiohttp）
- 现代化 GUI 界面（PyQt6）
- 数据处理（pandas + openpyxl）
- 跨平台支持（Windows/macOS/Linux）

### 已知限制
- 仅支持快手平台
- 需要稳定的网络连接
- 受平台反爬虫机制影响

## 版本说明

### 版本号格式
采用语义化版本号：`主版本号.次版本号.修订号`

- **主版本号**：不兼容的 API 修改
- **次版本号**：向下兼容的功能性新增
- **修订号**：向下兼容的问题修正

### 更新类型
- **新增**：新功能
- **修改**：对现有功能的修改
- **弃用**：即将移除的功能
- **移除**：已移除的功能
- **修复**：问题修复
- **安全**：安全相关的修复
